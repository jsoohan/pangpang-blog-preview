# PP Content GitHub Pages

Static HTML package for publishing PP Content Automation outputs.

## GitHub Pages setting

- Source: `Deploy from a branch`
- Branch: target branch
- Folder: `/pages`

## Current pages

- `/threadlifting/` — 실리프팅이란? 원리·종류·효과·부작용 정리
- `/threadlifting-vs-ulthera/` — 실리프팅 vs 울쎄라, 결국 무엇이 다를까?

## Naming convention

- 영문 슬러그·디렉토리·식별자는 **threadlifting** 으로 통일. `silifting`은 사용 금지 (잘못된 영문 표기).
- 한국어 본문·UI 라벨은 "실리프팅"으로 유지.

## Asset policy

- Uses static HTML/CSS only.
- Uses owned YouTube thumbnails from `@pangpangclinic` as remote images.
- Does not require LLM API keys or runtime server code.
